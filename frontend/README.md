# Interview Genius Frontend

React + Vite frontend for the Interview Genius recruitment platform.

## Environment

Create `frontend/.env` from `frontend/.env.example` and set:

- `VITE_NODE_API_URL`
- `VITE_PYTHON_API_URL`

## Run

```bash
npm install
npm run dev
```

The app expects the Node API and Python AI service to be running.
